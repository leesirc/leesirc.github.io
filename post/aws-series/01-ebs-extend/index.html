ebs简单认识

ebs

ebs, 即 elastic block storage

ebs extend

1.增加卷的大小不会自动增加该卷分区的大小, 扩展该卷分区到该卷的大小
2.扩展文件系统的大小

  1. 在aws console界面修改EBS卷的大小

  2. 登陆aws ec2, 扩展文件分区与文件系统

# 验证每个卷的文件系统
df -Th

# 检查该卷是否具有必须要扩展的分区
lsblk

# 在相应卷上扩展该分区到卷的大小
# nvme<0-26>n1 为nvme ebs卷; 其他类型为ebs卷
growpart /dev/nvme0n1 1

# 普通ebs卷
# growpart /dev/xvda 1

# 检验
lsblk
df -Th

# 扩展文件系统
## xfs系统进行扩展
## xfs_growfs -d /data

## ext4系统进行扩展
# nvme<0-26>n1 为nvme ebs卷; 其他类型为ebs卷
resize2fs /dev/nvme0n1p1
#resize2fs /dev/xvda1
